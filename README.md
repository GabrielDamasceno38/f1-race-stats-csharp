F1 Race Stats (C#)

C# learning project that calculates F1 drivers' total race time and fuel consumption, focusing on variables, types, and basic operators.

What it does
Simulates a 3-lap race with 5 drivers (Verstappen, Hamilton, Antonelli, Alonso, Leclerc), calculating:

Total race time (sum of lap times)
Total fuel consumption
Number of pit stops

Concepts practiced

Variables and data types (string, double, int)
Arithmetic operators
String interpolation
Output formatting with Console.WriteLine
